Steps for Scraping Amazon Website

1) We need to import some Python libraries like BeautifulSoup, requests, time, datetime, smtplib, csv and pandas

2) Connect to the Amazon webpage to connect.

3) For headers we need to get some information from our personal computer or laptop by using https://httpbin.org/get

4) By using BeautifulSoup extract the data from an HTML or XML document and then clean that data.

5) List can be made for header and data.

6) .csv file will be created and open for data entry and analysis.

7) Functions are defined for price check and sending emails if prices get lowered.
